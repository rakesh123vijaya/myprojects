# AI-Powered Customer Churn Analysis Dashboard

## 📊 Project Overview
This project analyzes customer churn patterns using Julius AI and visualizes business insights through an interactive Power BI dashboard.

## 🚀 Features
- AI-driven churn prediction
- Customer risk segmentation
- KPI dashboard
- Interactive visualizations
- Business intelligence insights

## 🛠 Tools Used
- Julius AI
- Power BI
- Data Analytics
- Predictive Analytics

## 📈 Dashboard Insights
- Churn analysis by contract type
- Customer risk levels
- Internet service churn trends
- Monthly charges vs churn
