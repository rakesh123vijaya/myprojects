# Retail Sales & Profit Analysis Dashboard using Excel

## Project Overview

This project is an interactive Excel dashboard built to analyze retail sales, profit, regional performance, product profitability, and salesperson performance.

## Features

* KPI Cards
* Pivot Tables
* Pivot Charts
* Interactive Slicers
* Monthly Sales Trend Analysis
* Conditional Formatting
* Dynamic Dashboard

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Design

## Dashboard Insights

* Region-wise sales analysis
* Product profitability analysis
* Salesperson performance tracking
* Monthly sales trends
